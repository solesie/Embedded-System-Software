# Embedded System Software [CSE4116]
## Target Board
• Freescale I.MX6Q Cortex-A9 Dual core  
• USB 2.0, SATA 1.0/2.0/3.0 Interface   
• MIPI(Mobile Industry Processor Interface)     
• EMMC 4.4 / T-FLASH    
• WiFi / Bluetooth / GPS    
• Ethernet 10/100M bps  
• HDMI mirror screen    
• 10 points touchscreen     
• Flexcan   
• Linux 3.10.17     
• Android Jellybean
